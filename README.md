# Contour-Based Object Analyzer

A Streamlit web application for detecting and analyzing shapes in images using computer vision.

**By: 23MIA1064 - SALAI NIMALAN**

## Features
- Upload images (PNG, JPG, JPEG)
- Automatic shape detection (Triangle, Square, Rectangle, Pentagon, Hexagon, Circle)
- Area and perimeter measurements
- Visual contour overlay
- Detailed analysis metrics

## Running Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
1. Push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repository
5. Set main file path to `app.py`
6. Deploy!
